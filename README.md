# NERDKit

NERDKit (or NK) is an opinionated "framwork" and collection of utilities used within
Nerd as a Service to create Go based web applications. Agree with it or not, this
helps us get stuff done.

## Our design rules:

1. This is not a framework, just a collection of things you CAN use, ergo:

   YOU OPT IN TO USE SOMETHING, NOTHING IS REQUIRED.
